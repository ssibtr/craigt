Website Advanced Wishlist

Recently, E-commerce rate has grown extensively. So, what is the reason for shopping online through an e-commerce website? Obviously because of its ease and convenience.
Apparently, various e-commerce platforms have provided many different features to their customer. Interestingly, the most trending feature nowadays is Wishlist. Basically, the Wishlist is a place where the customer can store the product which he/she wants to buy in the future.
Likewise, Odoo Advanced Wishlist allows a user to create the multiple wishlists (eg- Christmas eve Wishlist, Mother's day wishlist etc). Additionally, This module provides an easy interface for the user to add/remove products from multiple wishlists. The customer can simply click on "Add to wishlists" (a heart-shaped icon) option to add or remove the product from the wishlist in one click. Moreover, customers can manage multiple wishlists in one place.

Features:

  • This module allows the user to add product in his favourite wishlist in just one click.
  • User can add/remove same product from any wishlist by clicking on "Add to wishlist" (a heart-shaped icon).
  • While adding product user can see that in how many wishlists that paricular product is added.
  • Easy interface to manage multiple features of wishlists like create, delete, edit.
  • A user has the right to move the product from wishlist to cart or can move complete wishlist to cart in one go.
  • The product can be moved from one wishlist to other wishlists.

Wishlist page view

From this page customer can manage all his wishlists.

  • It's very simple for the customer to create, delete and edit wishlists.
  • It provides the customer with the feature of removing the products from wishlist.
  • Customer can either move products from any wishlist to the cart or move it from one wishlist to another.
  • Moreover, the customer can directly add a complete wishlist to the cart.
  • Customer can change his "Default Wishlist" without any effort.
